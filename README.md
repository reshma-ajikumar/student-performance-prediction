# Student Performance Prediction

This project predicts student performance using Machine Learning classification techniques.

## Dataset
The dataset contains student information such as study hours, attendance, and previous academic performance.

## Libraries Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model
Classification model (e.g., Logistic Regression / Decision Tree) used to classify student performance.

## Steps
1. Data preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature selection
4. Model training using classification algorithms
5. Model evaluation (accuracy, confusion matrix)

## Result
The model classifies student performance into categories (e.g., Pass/Fail or High/Medium/Low).
